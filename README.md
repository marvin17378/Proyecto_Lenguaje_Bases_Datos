# 🎬 CineNova - Proyecto de Lenguajes de Bases de Datos

## 📖 Descripción
Este es el repositorio de nuestro proyecto de Lenguajes de bases de datos; CineNova.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java
* **Base de Datos:** Oracle SQL

## ⚙️ Configuración y Ejecución
Para que el proyecto compile y funcione correctamente en tu entorno local, es estrictamente necesario configurar las dependencias manuales:

1. Clona este repositorio:
   bash
   https://github.com/marvin17378/Proyecto_Lenguaje_Bases_Datos.git
Abre el proyecto en tu entorno de desarrollo (IDE).

Paso crítico: Se deben agregar las librerías de la carpeta lib en Referenced Libraries.

Configura las credenciales y la conexión a la base de datos.

Compila y ejecuta la aplicación.
